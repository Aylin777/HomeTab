# HomeTab

Selfhosted life/home assistant.  
Merge of three projects:
- tasktab
- tasktab-android
- gotag

WIP: merge code of gotag

## Requirements to run

- x86-64 CPU (AMD/Intel)
- Linux (Windows support in the future)
- MariaDB / MySQL
- Redis
